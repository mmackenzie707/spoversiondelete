# spoversiondelete
PowerShell program that will clean up stale versions in Microsoft SharePoint Online

Requirements:

Powershell Version: 2.12.0 (If you have version 3.*, you will need to uninstall the version using / Uninstall-Module PnP.PoserShell -AllVersions -Force)
 -To install the correct version of PnP.PowerShell you will need to force PowerShell to install the correct version. Use this code, Install-Module PnP.PowerShell -RequiredVersion 2.12.0 -Force -Scope CurrentUser
